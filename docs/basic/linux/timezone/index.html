            设置时区 #

            CentOS 和 Ubuntu 的时区文件是 /etc/localtime，

            但是在 CentOS7 以后 localtime 变成了一个链接文件

            [root@centos7 ~]# ll /etc/localtime
lrwxrwxrwx 1 root root 33 Oct 12 11:01 /etc/localtime -> /usr/share/zoneinfo/Asia/Shanghai

            $ cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

            最好的方法是使用 timedatectl 命令

            $ timedatectl list-timezones |grep Shanghai    #查找中国时区的完整名称
Asia/Shanghai
$ timedatectl set-timezone Asia/Shanghai    #其他时区以此类推

            或者直接手动创建软链接

            $ ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
